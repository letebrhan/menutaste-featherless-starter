# Submission Notes

## Project Name

MenuTaste AI Agent

## Short Description

A Featherless-powered AI agent that helps food entrepreneurs evaluate food and drink ideas for nutrition, quality, risk, market fit, and launch readiness.

## Long Description

MenuTaste is a web-based enterprise AI agent for cafes, restaurants, caterers, cloud kitchens, and packaged food startups. Users enter a product idea and ingredients. The app validates the input, estimates nutrition signals, detects allergen and dietary risks, scores the product, calls Featherless for strategic reasoning, and generates recommendations plus a launch checklist.

## Technology Used

- Python
- Streamlit
- Featherless API
- OpenAI-compatible client
- Pydantic
- Pandas
- Docker
- Vultr VM
